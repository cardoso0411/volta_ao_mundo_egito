# Volta ao Mundo - Egito

Projeto desenvolvido para a disciplina de Desenvolvimento Web III. Este site apresenta um trabalho educativo sobre o Egito, com seções de história, turismo, cultura, galeria de imagens, curiosidades e um formulário de contato.

## Estrutura do projeto

- `index.html`
  - Página principal do site.
  - Contém o cabeçalho com navegação, seção hero, seções de conteúdo e rodapé.
- `styles.css`
  - Estilização customizada do site.
  - Complementa o Bootstrap usado para o layout e componentes.
- `assets/images/`
  - Pasta com imagens usadas nas seções de hero, turismo, galeria e outros blocos visuais.

## Conteúdo principal

- **Home**
  - Apresenta o tema do trabalho: a civilização egípcia.
  - Destaca história, cultura e turismo.
  - Inclui botões de navegação rápida.
- **História**
  - Resume o desenvolvimento do Egito Antigo.
  - Apresenta os períodos Pré-Dinástico, Antigo Império e Médio/Novo Império.
- **Turismo**
  - Mostra os principais pontos turísticos: Pirâmides de Gize, Esfinge, Rio Nilo e Vale dos Reis.
- **Cultura**
  - Fala sobre religião, costumes, culinária e arte/ escrita egípcia.
- **Galeria**
  - Exibe imagens de templos, deserto, barco no Nilo, hieróglifos, panorama, Cairo, museu e máscara de faraó.
- **Curiosidades**
  - Traz fatos rápidos sobre calendário solar, mumificação, medicina e engenharia.
- **Contato**
  - Formulário de contato com campos de nome, e-mail e mensagem.

## Tecnologias usadas

- HTML5
- CSS3
- Bootstrap 5.3 (via CDN)
- Google Fonts (`Montserrat` e `Playfair Display`)

## Como visualizar o site

1. Abra o arquivo `index.html` no navegador.
2. Alternativa: use um servidor local simples, se desejar.

## Observações

- O site é uma página estática e não possui backend.
- O formulário de contato é apenas visual; não há processamento de envio no projeto atual.
- O design usa cores e elementos inspirados no Egito, como tons de areia, dourado e marrom.
